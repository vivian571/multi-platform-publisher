# GitHub 提交命令

## 1. 查看当前状态
git status

## 2. 添加所有更改
git add .

## 3. 提交更改
git commit -m "更新微信发布工具和文档"

## 4. 推送到远程仓库
git push origin main

## 5. 如果遇到问题，可以强制推送（谨慎使用）
# git push -f origin main

## 6. 拉取远程最新更改（在推送前建议先拉取）
# git pull origin main

## 7. 查看提交历史
# git log --oneline

## 8. 查看远程仓库信息
# git remote -v
